1. Download and install Maven from http://maven.apache.org/download.html.
2. Modify your /etc/hosts file (c:\windows\system32\drivers\etc\hosts on 
   Windows) so demo.raibledesigns.com points to your local machine.
3. Run "mvn jetty:run-war -Djetty.port=80" (you'll need to sudo this on 
   *nix).
4. View the application at http://demo.raibledesigns.com/gwt-oauth

Please send any questions to matt@raibledesigns.com or search for 
"gwt oauth" and post as comments to blog entries on 
http://raibledesigns.com.
